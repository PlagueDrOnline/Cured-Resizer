🎭 Cured Resizer
The sterile solution for GitHub social previews.

In an ecosystem polluted by bloated image-editing suites and tracking-heavy online tools, Cured Resizer provides a localized, surgical remedy. This is a single-file, client-side utility designed to force-fit images into GitHub's ideal 1280x640 social preview dimensions.

🩺 The Diagnosis
GitHub social preview images are often cropped, stretched, or rendered poorly if not sized correctly. Existing tools are either bloated "monsters" or require you to upload your private data to a third-party server.

🧪 The Cure
Zero Telemetry: The processing happens entirely in your browser. No files are uploaded to any server.

Privacy-First: Your data stays on your machine.

Surgical Precision: Hard-coded to output 1280x640 JPEGs.

Zero-Bloat: No external dependencies, no tracking scripts, no frameworks—just pure, clinical HTML5 and Canvas.

⚙️ How to Operate
Open: Run index.html in any modern web browser.

Upload: Drop your raw image into the terminal box.

Adjust: Set your desired quality (0.1 - 1.0).

Inject: Click "Inject Resize & Download" to receive your optimized, GitHub-ready asset.
